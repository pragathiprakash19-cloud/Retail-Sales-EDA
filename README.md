📊 Retail Sales Data Analysis (EDA)
📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to understand sales performance, customer behavior, and product trends.
The analysis is carried out using Python, focusing on data cleaning, statistical analysis, and visualization to extract meaningful business insights.

🛠️ Technologies Used
-Python
-Pandas
-Seaborn
-Matplotlib

⚙️ Features
-Data loading and inspection
-Data cleaning (handling missing -values and duplicates)
-Date-time processing and feature extraction (Month)
-Descriptive statistical analysis
-Time series analysis of sales
-Customer analysis based on gender
-Product category analysis
-Data visualization and correlation analysis

📂 Dataset
The dataset used in this project contains the following columns:
-Date – Transaction date
-Product Category – Category of product
-Sales – Sales amount
-Gender – Customer gender

📊 Visualizations
The project includes:
-Line chart for Sales Trend Over Time
-Line chart for Monthly Sales Trend
-Bar chart for Sales by Gender
-Bar chart for Sales by Product Category
-Histogram for Sales Distribution
-Heatmap for Correlation Analysis

🔍 Key Insights
-Sales trends can be clearly observed over time
-Certain product categories contribute more to total sales
-Customer purchasing behavior varies based on gender
-Monthly analysis helps identify high and low sales periods

▶️ How to Run the Project
Install required libraries:
pip install pandas matplotlib seaborn

Run the Python script:
python retail_eda.py
