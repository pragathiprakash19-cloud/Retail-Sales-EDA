# 📊 Retail Sales Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on retail sales data to understand sales trends, customer behavior, and product performance.


## 🛠️ Technologies Used

- Python  
- Pandas  
- Seaborn  
- Matplotlib  


## ⚙️ Features

- Data loading and inspection  
- Data cleaning (handling missing values and duplicates)  
- Time series analysis  
- Customer analysis by gender  
- Product category analysis  
- Data visualization  

## 📂 Dataset

- Date – Transaction date  
- Product Category – Type of product  
- Sales – Sales amount  
- Gender – Customer gender  


## 📊 Visualizations

- Sales Trend Over Time  
- Monthly Sales Trend  
- Sales by Gender  
- Sales by Product Category  
- Sales Distribution  
- Correlation Heatmap  


## 🔍 Key Insights

- High-performing product categories identified  
- Sales trends observed over time  
- Customer behavior varies by gender  

## ▶️ How to Run

```bash
pip install pandas matplotlib seaborn
python retail_eda.py
```
